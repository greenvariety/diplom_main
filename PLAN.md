## Правило: как добавлять ошибки в новые поля

Каждое обязательное поле должно работать так: при ошибке - красная рамка + красный текст под полем, оба исчезают через 15-20 секунд.

### Шаблон для input / select / textarea:

```jsx
<Field label="Название поля" required error={условие_ошибки ? 'Текст ошибки' : null}>
  <input className={`input ${условие_ошибки ? 'is-error' : ''}`} ... />
</Field>
```

- Для `select` - `className={`select ${...}`}`
- Для `textarea` - `className={`textarea ${...}`}`
- `Field` автоматически показывает текст ошибки через `FadingError` (исчезает сам)
- CSS автоматически убирает красную рамку когда текст исчезает - ничего дополнительно делать не надо

### Пример с touched-состоянием (валидация при вводе):
```jsx
const errs = {};
if (!vals.name.trim()) errs.name = 'Обязательно';

<Field label="Имя" required error={touched.name && errs.name}>
  <input className={`input ${touched.name && errs.name ? 'is-error' : ''}`}
    onBlur={() => setTouched(t => ({ ...t, name: 1 }))} ... />
</Field>
```

### Пример с одним err-состоянием (валидация при сабмите):
```jsx
<Field label="Название" required error={err && !name.trim() ? err : null}>
  <input className={`input ${err && !name.trim() ? 'is-error' : ''}`}
    onChange={e => { setName(e.target.value); setErr(''); }} ... />
</Field>
```
