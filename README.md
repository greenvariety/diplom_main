# АИС учёта студентов колледжа

Автоматизированная информационная система для управления студентами, сотрудниками, группами и документами учебного заведения.

---

## Шаг 1 - Установите Python

Скачайте Python 3.11 по прямой ссылке:

**https://www.python.org/ftp/python/3.11.9/python-3.11.9-amd64.exe**

При установке обязательно поставьте галочку **"Add Python to PATH"** (внизу окна установщика), затем нажмите Install Now.

---

## Шаг 2 - Установите проект

Дважды кликните файл **`1_Установить.bat`**

Откроется консоль. Скрипт автоматически:
- Установит все необходимые библиотеки
- Создаст базу данных
- Создаст учётную запись для входа

Дождитесь надписи **"Установка завершена успешно!"** и закройте консоль.

---

## Шаг 3 - Запустите проект

Дважды кликните файл **`2_Запустить.bat`**

Браузер откроется автоматически. Если не открылся - перейдите вручную по адресу:

**http://127.0.0.1:8000**

Для остановки сервера закройте консольное окно.

---

## Данные для входа

Логин и пароль выводятся в консоли при выполнении шага 2.
