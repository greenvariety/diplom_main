АИС управления контингентом учебного заведения

Веб-приложение для учёта студентов, сотрудников, групп, факультетов и документов.
Поддерживает три роли: владелец, администратор, преподаватель.


Установка

1. Установите Python 3.11: https://www.python.org/ftp/python/3.11.9/python-3.11.9-amd64.exe
   При установке отметьте "Add Python to PATH".

2. Запустите Установить зависимости.bat
   Установит все пакеты, создаст базу данных и аккаунт администратора.
   Логин и пароль появятся в консоли.

3. Запустите Запустить проект.bat
   Откроется браузер на http://127.0.0.1:8000


Email

Чтобы заработала отправка кода на почту, заполните в env/.env:

    EMAIL_HOST_USER=ваш@gmail.com
    EMAIL_HOST_PASSWORD=пароль-приложения-google

Если не заполнить, код подтверждения не отправится на почту, а появится
в консоли сервера (в окне терминала где запущен проект).


Структура проекта

│   ├── urls.py           корневые URL (подключает core/urls.py)
│   ├── asgi.py / wsgi.py точки входа для сервера
│
├── core/                 единственное Django-приложение
│   ├── models.py         все модели БД
│   ├── urls.py           все /api/... маршруты + catch-all -> фронтенд

│   ├── utils.py          log_action, email-хелперы, декораторы ролей

│   │
│   ├── api_auth.py       логин, регистрация, восстановление пароля
│   ├── api_profile.py    профиль пользователя (смена пароля, email, логина)
│   ├── api_organizations.py  организации (CRUD, переключение)
│   ├── api_faculties.py  факультеты

│   ├── api_employees.py  сотрудники, аккаунты, назначение предметов

│   ├── api_positions.py  должности
│   ├── api_subjects.py   предметы
