Шаг 1 - Установите Python 3.11
https://www.python.org/ftp/python/3.11.9/python-3.11.9-amd64.exe
При установке отметьте "Add Python to PATH"

Шаг 2 - Запустите Установить зависимости.bat
Дождитесь надписи "Установка завершена успешно!"
Логин и пароль будут выведены в консоли.

Шаг 3 - Запустите Запустить проект.bat
Браузер откроется автоматически на http://127.0.0.1:8000
