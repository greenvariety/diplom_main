# Проект: АИС учебного заведения

## Что это

Автоматизированная информационная система (АИС) для управления учебным процессом в колледже/университете. Дипломная работа.

## Цель

Заменить бумажный/Excel-учёт на веб-систему с разграничением прав доступа, историей изменений и документооборотом.

## Аудитория

| Роль | Кто | Что делает |
|---|---|---|
| Владелец (owner) | Директор / ИТ-отдел | Полный доступ, управление организациями и пользователями, одобрение удалений |
| Администратор (admin) | Секретариат / деканат | CRUD студентов, сотрудников, групп, факультетов |
| Преподаватель (teacher) | Преподаватели | Просмотр своих групп и студентов (read-mostly) |

## Контекст

- Проект является **дипломной работой**, поэтому приоритет — полнота функционала и документация, а не production-ready код.
- Локальная разработка, запуск через `.bat`-файл на Windows.
- База данных — SQLite (достаточно для диплома, не нужна PostgreSQL).
