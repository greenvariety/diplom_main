## Чеклист ручного тестирования

### 1. Авторизация
- [ ] Регистрация нового owner (с подтверждением кода на email)
- [ ] Повторная отправка кода
- [ ] Вход по логину/паролю
- [ ] Восстановление пароля через email
- [ ] Выход из системы

### 2. Организации
- [ ] Создать организацию
- [ ] Переименовать организацию
- [ ] Удалить организацию (с вводом кода на email)
- [ ] Переключиться между несколькими организациями

### 3. Дашборд
- [ ] Видна статистика (owner)
- [ ] Виден счётчик заявок на удаление
- [ ] Видны последние логи

### 4. Факультеты
- [ ] Создать факультет
- [ ] Переименовать
- [ ] Открыть детальную страницу (список групп)
- [ ] Подать заявку на удаление (admin), одобрить (owner)

### 5. Группы
- [ ] Создать группу (автогенерация номера)
- [ ] Редактировать
- [ ] Открыть детальную страницу (студенты, предметы)
- [ ] Добавить/убрать предмет в группе
- [ ] Удалить группу

### 6. Студенты
- [ ] Создать студента
- [ ] Редактировать данные
- [ ] Привязать опекуна
- [ ] Перевести в другую группу
- [ ] Пройти весь цикл статусов: на рассмотрении → зачислен → отчислен
- [ ] Загрузить документ (фото/PDF)
- [ ] Экспорт в Excel
- [ ] Фильтры: по факультету, группе, статусу, поиск

### 7. Опекуны
- [ ] Создать опекуна
- [ ] Привязать студента
- [ ] Открыть детальную страницу

### 8. Сотрудники
- [ ] Создать сотрудника с фото
- [ ] Назначить должность
- [ ] Назначить предмет в группе
- [ ] Открыть детальную страницу
- [ ] Кликнуть на должность - переход к отфильтрованному списку

### 9. Должности
- [ ] Создать, переименовать, удалить

### 10. Предметы
- [ ] Создать предмет
- [ ] Открыть детальную страницу (преподаватели, группы)

### 11. Пользователи (owner)
- [ ] Создать admin/teacher
- [ ] Привязать к сотруднику
- [ ] Сменить пароль

### 12. Заявки на удаление
- [ ] Создать заявку (через admin)
- [ ] Одобрить с паролем (через owner)
- [ ] Отклонить заявку

### 13. Журнал изменений
- [ ] Видны записи, фильтр по пользователям

### 14. Права доступа
- [ ] Войти как teacher - проверить что видит только свои группы
- [ ] Войти как admin - нет доступа к журналу и заявкам
