# Функции системы

## Реализовано

### Авторизация
- [x] Вход по логину и паролю (`/api/auth/login/`)
- [x] Выход (`/api/auth/logout/`)
- [x] Регистрация нового owner с подтверждением email (6-символьный код)
- [x] Повторная отправка кода регистрации
- [x] Восстановление пароля через email-код
- [x] JWT: access (60 мин) + refresh (30 дней), автообновление в axios-клиенте
- [x] Кастомная модель пользователя с ролями (owner / admin / teacher)

### Организации (Institution)
- [x] Создание, редактирование, удаление организации (owner)
- [x] Переключение между организациями (owner с несколькими)
- [x] Удаление организации с подтверждением email-кодом
- [x] Выбор организации после входа (OrgPickerScreen)

### Факультеты (`/api/faculties/`)
- [x] Список с поиском
- [x] Создание, редактирование (admin+)
- [x] Детальная страница (список групп факультета)
- [x] Заявка на удаление / прямое удаление (owner)

### Группы (`/api/groups/`)
- [x] Список с поиском (преподаватель видит только свои)
- [x] Создание, редактирование (admin+)
- [x] Детальная страница (студенты, предметы)
- [x] Добавление/удаление предметов в группе
- [x] Заявка на удаление / прямое удаление (owner)
- [x] Автогенерация номера группы при создании

### Студенты (`/api/students/`)
- [x] Список с фильтрами (поиск, факультет, группа, статус)
- [x] Создание, редактирование (admin+)
- [x] Детальная страница (данные, опекуны, документы, история)
- [x] Статусы: на рассмотрении → на зачисление → зачислен → на отчисление → отчислен / переведён
- [x] Перевод студента в другую группу/факультет
- [x] Привязка/отвязка опекунов
- [x] Заявка на удаление / прямое удаление (owner)
- [x] Экспорт в Excel

### Опекуны/Родители (`/api/parents/`)
- [x] Список с поиском (admin+)
- [x] Создание, редактирование
- [x] Детальная страница (связанные студенты, документы)
- [x] Привязка студентов к опекуну
- [x] Заявка на удаление / прямое удаление (owner)

### Сотрудники (`/api/employees/`)
- [x] Список с поиском (admin+)
- [x] Создание, редактирование (с фото)
- [x] Детальная страница (группы, предметы, документы)
- [x] Назначение предметов в группы
- [x] Заявка на удаление / прямое удаление (owner)

### Должности (`/api/positions/`)
- [x] CRUD (owner only)

### Предметы (`/api/subjects/`)
- [x] CRUD (admin+)
- [x] Детальная страница (какие преподаватели ведут, в каких группах)

### Документы
- [x] Загрузка файлов для студента/сотрудника/опекуна
- [x] Просмотр (изображения отображаются inline)
- [x] Удаление (admin+)
- [x] Типы: паспорт, СНИЛС, полис ОМС, аттестат, приказ, прочее

### Управление пользователями (`/api/users/`)
- [x] Список, создание, редактирование (owner only)
- [x] Смена пароля
- [x] Привязка пользователя к сотруднику

### Заявки на удаление (`/api/delete-requests/`)
- [x] Список ожидающих заявок (owner)
- [x] Одобрение с вводом пароля (физически удаляет объект)
- [x] Отклонение
- [x] Счётчик ожидающих заявок (`/api/delete-requests/count/`)

### Журнал изменений (`/api/audit-log/`)
- [x] Последние записи (owner), фильтр по пользователям
- [x] Поля: кто, что сделал, с каким объектом, данные до/после

### Дашборд (`/api/dashboard/`)
- [x] Для owner: статистика + последние логи + счётчик заявок
- [x] Для admin: общая статистика
- [x] Для teacher: свои группы и предметы
